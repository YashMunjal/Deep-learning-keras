# Tensorflow-with-keras 
A Practical Guide to Machine Learning with TensorFlow 2.0 &amp; Keras course by  Vadim Karpusenko FEM
# Prerequisite
- Python
- Jupyter Notebook
# Installation

- Install dependencies
        ``pip install -r requirements.txt``
- Run jupyter notebook
        ``jupyter notebook``
